## Лабораторная работа. Chroma key
Chroma key: Заданы два изображения и цвет (интервал задается пользователем). Необходимо заменить все соответствующие заданному цвету области одного изображения на второе изображение.

## Цель работы
Научиться реализовывать один из простых алгоритмов обработки изображений.

## Задание

1. Разобрать теоретическую базу задания, принцип работы алгоритма
2. Реализовать базовый алгоритм самостоятельно на Python.
3. Реализовать программу Chroma key с использованием встроенных функций какой-либо библиотеки (OpenCV, PIL и др.). 
4. Сравнить быстродействие реализованных вариантов.
